  2) HOW TO PLAY
___________________________

1) Launch helloreaper.exe.
2) Kill all enemies while dodging the attacks to beat each level.
_____________________________________________________________________________________
 
  3) CONTROLS
___________________________
*Note that we highly recommend using GamePad!*

Gamepad:
Move		- Left Stick
Move 1 Tile	- Left trigger + Left stick
Attack		- Right Stick
Select		- A
Restart		- Y
Pause		- Start
Back To Main Menu - B while the game is PAUSED.

Keyboard:
Move		- WASD keys
Move 1 tile	- Shift + WASD keys
Attack		- Arrow keys
Select		- Space
Restart		- R
Pause		- P or Esc
Back To Main Menu - M while the game is PAUSED.
